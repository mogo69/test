Работа с украинским бизнесом, работа с лучшими представителями, работа с теми, кто показывает пример остальным.

Работа с широкой аудиторией. Мы готовим образовательные курсы по сервису для представителей разных профессий + планируем создание инновационного центра, где будут проводиться разработки проектов по улучшению сервиса для конкретных украинских компаний.
